**Парадигма программирования определяется**:
- вычислительной моделью
- базовыми программными единицами
- методами разделения абстракций

К **основным парадигмам** программирования относятся:
- императивное программирование (пошаговые инструкции)
- структурное программирование (блоки кода)
- функциональное программирование (чистые функции)
- объектно-ориентированное программирование (объекты, классы)
- логическое программирование (факты, правила)

Язык программирования не обязательно использует только одну парадигму. Существуют **мультипарадигменные** языки. Создатели таких языков считают, что ни одна парадигма не может быть одинаково эффективной для всех задач и следует позволять программисту выбирать лучшую парадигму для решения каждой задачи. 

*Python* — мультипарадигменный язык. Он поддерживает императивное, структурное, функциональное, объектно-ориентированное и другие стили программирования.

Почитать подробнее о парадигмах программирования можно по [ссылке](https://ru.wikipedia.org/wiki/Парадигма_программирования).
