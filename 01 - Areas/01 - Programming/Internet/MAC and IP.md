# MAC
A **MAC address** (Media Access Control address) is a unique hardware identifier assigned to a device's network interface controller (NIC) by the manufacturer. Its primary purpose is to enable devices to be _uniquely identified_ within a local network, ensuring data sent across that network reaches the correct destination.

> [!note] Unique hardware ID for a device on a local network


# IP
An **IP address** (Internet Protocol address) is a unique identifier assigned to a device on a network to specify its location for network communication, allowing it to send and receive data across the internet or local networks.

# Difference
- **IP** -> wider network, **MAC** -> local area network (LAN).
- **IP** -> Layer 3, **MAC** -> Layer 2.
- **IP addresses can be public or private** (assigned by an ISP or router), and may change as the device moves or reconnects; **MAC addresses are assigned by the device manufacturer** and rarely change unless manually altered.
