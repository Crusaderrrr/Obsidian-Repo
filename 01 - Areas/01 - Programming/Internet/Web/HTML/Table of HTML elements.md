| Element        | Description                                                                                     |
| -------------- | ----------------------------------------------------------------------------------------------- |
| `<html>`       | The root element that wraps the entire HTML document.                                           |
| `<head>`       | Contains metadata, title, links to stylesheets, and scripts.                                    |
| `<title>`      | Sets the title of the document (shown in browser tab).                                          |
| `<meta>`       | Defines metadata such as character encoding, viewport, and SEO information.                     |
| `<link>`       | Links external resources like CSS files or icons.                                               |
| `<script>`     | Embeds or references JavaScript code.                                                           |
| `<body>`       | Contains all the visible content of the web page.                                               |
| `<header>`     | Denotes introductory content or a group of navigation links, typically at the top of a section. |
| `<nav>`        | Marks up navigation menus or links to other pages/sections.                                     |
| `<main>`       | Specifies the main content unique to the document.                                              |
| `<section>`    | Defines thematic grouping of content, often with a heading.                                     |
| `<article>`    | Represents a self-contained piece of content, like a blog post or news article.                 |
| `<aside>`      | Contains content tangentially related to the main content (e.g., sidebars, pull quotes).        |
| `<footer>`     | Defines the footer for a section or page, often with copyright or contact info.                 |
| `<h1>`–`<h6>`  | Headings, with `<h1>` as the most important and `<h6>` the least.                               |
| `<p>`          | Paragraph of text.                                                                              |
| `<a>`          | Hyperlink to another page or resource.                                                          |
| `<ul>`         | Unordered (bulleted) list.                                                                      |
| `<ol>`         | Ordered (numbered) list.                                                                        |
| `<li>`         | List item, used within `<ul>` or `<ol>`.                                                        |
| `<img>`        | Embeds an image. Requires `src` and `alt` attributes.                                           |
| `<figure>`     | Groups media content (like images) with an optional caption.                                    |
| `<figcaption>` | Provides a caption for content inside `<figure>`.                                               |
| `<table>`      | Defines a table.                                                                                |
| `<tr>`         | Table row.                                                                                      |
| `<td>`         | Table cell (data).                                                                              |
| `<th>`         | Table header cell.                                                                              |
| `<form>`       | Collects user input via form controls.                                                          |
| `<input>`      | Accepts user input; type specified by the `type` attribute.                                     |
| `<textarea>`   | Multi-line text input.                                                                          |
| `<button>`     | Clickable button.                                                                               |
| `<select>`     | Drop-down list.                                                                                 |
| `<option>`     | Option in a drop-down list.                                                                     |
| `<label>`      | Defines a label for a form element.                                                             |
| `<div>`        | Generic container for flow content, useful for layout and grouping.                             |
| `<span>`       | Generic inline container for phrasing content.                                                  |
| `<br>`         | Line break.                                                                                     |
| `<hr>`         | Thematic break (horizontal rule).                                                               |
| `<strong>`     | Indicates strong importance (usually bold).                                                     |
| `<em>`         | Indicates emphasized text (usually italic).                                                     |
| `<code>`       | Marks up inline code snippets.                                                                  |
| `<pre>`        | Preformatted text, preserves whitespace and line breaks.                                        |
| `<details>`    | Creates a collapsible section the user can expand or collapse.                                  |
| `<summary>`    | Provides a visible heading for a `<details>` element.                                           |
| `<audio>`      | Embeds audio content.                                                                           |
| `<video>`      | Embeds video content.                                                                           |
| `<source>`     | Specifies media resources for `<audio>` and `<video>`.                                          |
| `<canvas>`     | Provides a drawing area for graphics via JavaScript.                                            |
| `<svg>`        | Embeds scalable vector graphics.                                                                |
| `<time>`       | Represents a specific time or date.                                                             |
| `<mark>`       | Highlights or marks text.                                                                       |
| `<progress>`   | Displays the progress of a task.                                                                |
| `<output>`     | Represents the result of a calculation or user action.                                          |
| `<dialog>`     | Defines a dialog box or modal window.                                                           |
