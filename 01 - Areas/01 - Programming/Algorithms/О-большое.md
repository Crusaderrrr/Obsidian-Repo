О-большое описывает насколько быстро работает алгоритм. Оно используется не для того, чтобы показать какое время будет затрачено, а для того, чтобы *сравнить количество операций*. 

>[!info] Оно указывает насколько быстро возрастает время выполнения алгоритма.

**Например**: 
- Время выполнения О-большое для линейного поиска - О(n)
- Время выполнения для бинарного поиска -  О($\log_{2}(n)$)
