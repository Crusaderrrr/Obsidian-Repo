*Сортировка выбором* - это алгоритм сортировки набора данных по неубыванию или по невозрастанию. 

При сортировке **по неубыванию** из неотсортированных элементов выбирается **минимальный**, а при сортировке **по невозрастанию** — **максимальный** и ставится на первое место массива, затем, в зависимости от направления сортировки ставится больший или меньший элемент и так далее.

**Шаги сортировки**:
1. среди всех элементов набора выбирает минимальный и ставит его на **первое** место
2. среди всех элементов набора, кроме первого, снова выбирает минимальный и ставит его на **второе** место
3. среди всех элементов набора, кроме первых двух, вновь выбирает минимальный и ставит его на **третье** место
4. аналогичным образом алгоритм ставит подходящие элементы на четвертое, пятое и остальные места

```python
def selection_sort (data) :
	n = len (data)
	
	for i in range(n - 1):
		index_min = i
		
		for j in range(i + 1, n):
			if data[j] < data[index_min]:
			index_min = j
			
		data[i], data[index_min] = data[index_min], data[i]
```

**Временнáя сложность алгоритма сортировки выбором** равна `O(n²)`
