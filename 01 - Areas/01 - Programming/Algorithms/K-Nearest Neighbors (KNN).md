This is an algorithm that is used in two ways: 
- **Classification** - category distribution 
- **Regression** - preventing the answer (in a number expression)

The first one is used in order to recommend something to users. Like a Netflix films. It uses the reviews of the user with similar favours and recommends films or series that other liked. 

The second one is used to prevent the answer, for example a bakery need to know how many pans they need to prepare the exact day.
