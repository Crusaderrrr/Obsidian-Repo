Это один из способов с помощью [[DFS, Depth-First Search|деревьев]] создать алгоритм, который, например, будет декодировать слово из его двоичнооо представления с помощью кодировок(Unicode, ASCII, etc) в буквы.

Выглядит это следующим образом: 
![[IMG_1435.jpeg]]
Код буквы а в двоичном виде - 010 (лево, право, лево)

**Для чего предназначен**: 
Для того, чтобы сократить количество *бит* необходимых для кодировки одной буквы (8 по стандарту). 

При этом **очень важно**, что в коде Хаффмана буквы располагаются только в *листовых узлах*. 
Для того, чтобы не было цикличности и ошибок.
