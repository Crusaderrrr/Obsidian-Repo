**Сортировка вставками** — это алгоритм сортировки набора данных по неубыванию или по невозрастанию. 

Идея алгоритма сортировки вставками заключается в разделении сортируемого набора на **отсортированную** и **неотсортированную** части. Изначально отсортированная часть включает в себя только первый элемент набора, а неотсортированная — все остальные. Затем алгоритм поочередно перемещает все элементы из неотсортированной части в отсортированную так, чтобы **отсортированная часть оставалась отсортированной**.

**Классическая реализация** сортировки вставками — это алгоритм, выполняющий сортировку списка **по неубыванию** путем его **изменения**. Данная реализация представлена на картинке. 

При сортировке набора данных длины `n` алгоритм сортировки вставками сначала ищет подходящую позицию для первого неотсортированного элемента, сравнивая его с одним отсортированным. Затем алгоритм ищет нужную позицию для второго неотсортированного элемента, сравнивая его с двумя отсортированными. В худшем случае поиск подходящей позиции требует сравнения неотсортированного элемента со всеми отсортированными, поэтому общее количество сравнений может быть пропорционально **квадрату** количества элементов в наборе.

Таким образом, **временнáя сложность алгоритма сортировки вставками** равна `O(n²)`, где `n` — длина набора данных.
