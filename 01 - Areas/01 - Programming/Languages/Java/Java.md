Java was designed to be simple, robust, and portable, with the philosophy of "write once, run anywhere" (WORA), meaning Java programs can run on any device with a compatible Java Virtual Machine (JVM) without modification.
- **Statically typed**, all the variables' types are declared explicitly. 
- **Strongly typed**, prevents unsafe conversions and limits type methods.
- **Object-oriented**, [[OOP]]
- **Class-based**, does not support prototype based programming (JS)
- **Concurrent**, multithreaded
- **Memory-safe**, garbage collector, reduces error with pointer corruptions, etc.
- **Compilation & Interpretation**:
	1. The source files `.java` are first compiled by java compiler (*javac*) into *bytecode* `.class` files. This bytecode is not machine-specific native code but a platform-independent instruction set.
	2. When running the program *Java Virtual Machine* (JVM) either compiles or interprets this *bytecode* depending on specific platform where it runs.
	3. Uses *JIT* (Just-In-Time) compiler and caches the results if necessary.
