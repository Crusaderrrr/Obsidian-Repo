These are bundles for developers of java.

There are 3 of them:
- **Java ME** (Mobile, embedded) -> *Java Micro Edition*
- **Java SE** (Desktop, Servers)-> *Java Standard Edition*
- **Jakarta EE** (Enterprise Apps) -> stands for *Jakarta Enterprise Edition* (previously Java EE), extends Java SE. Is opensource.
  
They provide us with basic util packages like `java.lang`, `java.util`, `java.io`.
