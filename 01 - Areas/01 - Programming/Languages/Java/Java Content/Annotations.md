Annotations in Java are **used for:**
- Give instructions for Java compiler
- Give instructions of analyzers of the code
- Give metadata to frameworks, etc.

These are annotations in Java:
- [[@Component]]
- [[@Autowired]]
- [[@Qualifier]]
- [[@Value]]
- [[@Scope]]
- [[@PostConstruct & @PreDestroy]]
- [[@Transactional]]
- [[@Configuration]]
- [[@ModelAttribute]]
