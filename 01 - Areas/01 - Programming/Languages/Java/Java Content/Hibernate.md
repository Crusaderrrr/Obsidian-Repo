This is an **implementation of JPA** and it follows ORM strategies. Consequently it is a **JPA Provider**, which means that it provides some database logic features, such as autocreation of tables.

The features are:
- Object Relational Mapping with Java objects 
- **HQL** - Hibernate Query Language 
- Caching 
- Transaction management
- Others...
