1. **Autoboxing**
	- This is the process of auto-converting the value type from primitive type to a wrapper class of this primitive type. 
	- *Example*: from int -> Integer; double -> Double
2. **Unboxing**
	- Reverse operation

This is automated process, can be executed by simply assigning the value of primitive type to a wrapper class or vice versa.
