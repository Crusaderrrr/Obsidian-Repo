Раньше люди не могли позволить себе покупать одну единицу техники на каждого рабочего поэтому была придумана **виртуализация** - это процесс при котором на одном устройстве могут работать несколько человек независимо друг от друга и развёртывать свои проекты которые не будут влиять на среду разработки других пользователей. 

Это работает следующим образом есть несколько типов *гипервизоров* - это программы которые контролируют эти самые среды для каждого пользователей: 
- 1й тип - гипервизор с драйверами для каждого компонента 
- 2й тип - гипервизор который работает вместе с ОС
- Гибридный - два предыдущих вместе 

На данный момент виртуализация это когда большая компания у которой есть свой сервер выделяет тебе место на этом самом сервере, так называемое *виртуальное пространство* которое ты можешь настраивать как хочешь, засчёт того, что это виртуальное пространство представляет собой полностью автономное ОС и среду разработки.
