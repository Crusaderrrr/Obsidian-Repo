Google's RPC framework that is used in microservices (enables interactions between them).

## Key features

- The **HTTP 2** is used here and it is 10-15% faster then his ancestor HTTP 1/1.
- Simple and fast data streaming 
- Instead of JSON, [[Protobuf]] is used.
- Code generation is present
- Comfortable procedure call

## Content 

- In `.proto` files we describe the endpoints and the object's structure they manipulate
