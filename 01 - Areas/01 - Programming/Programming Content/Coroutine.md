- This is `Сопрограмма` in Russian.
- A coroutine is a **suspendable computation** — it can **pause execution (suspend)** at certain points without blocking the underlying thread and **resume** later when the needed result is ready. This is made possible through **suspending functions**, which are marked with the `suspend` keyword.
