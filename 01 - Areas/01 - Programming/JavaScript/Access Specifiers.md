## Private
This is a keyword that allows to create a private the class that could not be accessed or modified manually, only via **getters and setters**.

Getters and Setters are special methods defined inside the class that work with private variables.
- Setters - modification
- Getters - accessing 

---
## Public
Can be accessed from **anywhere**.
Provides a universal access to a method, class or variable.

---
## Protected
Makes class members available to other classes **in the same package** and all **subclasses**, regardless of their package. This helps with encapsulation and controlled sharing within a class hierarchy.
