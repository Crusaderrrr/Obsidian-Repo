*Object Id* - 6813d7f5829072838b25feb1

There are 12 bytes:
- 4 bytes: timestamp
- 3 bytes: machine identifier 
- 2 bytes: process identifier
- 3 bytes: counter

So, we can have 16 million ID's

It is generated by MongoDB driver

We can manipulate ID with **mongoose**:
```JavaScript
const mongoose = require('mongoose');

const id = new.mongoose.Types.ObjectId();
console.log(id.getTimeStamp); // time of creation of the id

const isValid = mongoose.Types.ObjectId.isValid('1234');
console.log(isValid); // false
```

### Joi ObjectID
run `npm i joi-objected` 
We can change the validation function in `rental.js` as a model.
```JavaScript
Joi.objectid = require('joi-objectid')('Joi'); // importing new module 

...

const validateRental = rental => {
  const schema = Joi.object({
    carId: Joi.objectid().required(),
    customerId: Joi.objectid().required()
  });
  return schema.validate(rental);
};
```

**The better approach**:
Import `Joi.objectid = require('joi-objectid')('Joi');` in `index.js`, so there will be no need of importing it in every model file.

And then we can use it wherever we want:
`car.js`
```JavaScript
const validateCar = car => {
  const schema = Joi.object({
    modelName: Joi.string().required().min(5).max(50),
    companyId: Joi.objectId().required(), // new method
    numberInStock: Joi.number().required().min(0).max(25),
    dailyRentalRate: Joi.number().required().min(0).max(25)
  });
```
