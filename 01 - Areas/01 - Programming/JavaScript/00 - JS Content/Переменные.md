#JavaScript 

**Имена переменных:**
 - Типы и классы -> PascalCase
 - Константы -> DB_PASSWORD
 - Остальные переменные -> camelCase *(самый часто используемый)*
 
 **Объявление переменных**:
- `var`: 
	**Function-scoped**, can be re-declared and updated, *hoisted* and initialized as `undefined`
    
- `let`: 
	**Block-scoped**, can be updated but not re-declared in the same scope, *hoisted but not initialized* (`Reference error`).
    
- `const`:
	**Block-scoped**, cannot be updated or re-declared, must be initialized at declaration, *hoisted but not initialized* (`Reference error`).

**Hoisting** - moving a variable or a function to the top of their scope before the code execution.
