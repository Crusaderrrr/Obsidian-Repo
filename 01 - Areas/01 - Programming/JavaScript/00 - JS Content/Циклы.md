#JavaScript 

*for:* 
```JS
for (Начальная инструкция; Условие; Итерационное действие) {
	// блок кода выполняемый на каждой итерации
}
```
	
	Пример:
```JS
for (let i = 0; i < 5; i++) {
console.log(i)
} 
```

*while:*
```JS
while (Условие) {
	// Блок кода, выполняемый пока УСЛОВИЕ верно
}
```
	
	Пример:
```JS
let i = 0

while (i < 5) {
	console.log(i)
	i++
}
```

*do while:*
```JS
do {
	// блок кода выполняемый на каждой итерации
} while (Условие)
```
  - нужен для того чтобы выполнить блок кода хотя бы 1 раз

*for in:*
```JS
for (key in Object) {
	// действия с каждым свойством объекта
	// значение свойства - Object[key]
}
```
	
	Пример:
```JS
const myObject = {
	x: 10, 
	y: true,
	z: 'abc'
}

for (const key in myObject) {
	console.log(key, myObject[key])
}
```
	
	Пример со списком:
```JS
const myArray = [true, 10, 'abc', null]

for (const key in myArray) {
	console.log(myArray[key])
}
```

**Методы Object:**
- `Object.keys(myObject)` -> возвращает список ключей объекта
- `Object.values(myObject)` -> возвращает список значений объекта

*for of:*
```JS
for (Element of Iterable) {
 // Действия с элементом
}
```
