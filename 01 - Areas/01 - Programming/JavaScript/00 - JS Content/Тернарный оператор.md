#JavaScript 

*Структура*:
`Условие ? Выражение 1 : Выражение 2` - выражение 
- условие - любое выражение  
- если условие **правдиво** -> возвращается результат **выражения 1**
- если условие **ложно** -> возвращается результат **выражения 2**

*Рекомендованная структура:* 
```JS
Условие
	? Выражение 1
	: Выражение 2
```

*Примеры:*
```JS
const value1 = 11
const value2 = 24

value1 && value2 
	? myFunction(value1, value2)
	: myFunction2()
```

*Также можно:*
```JS
let value = 11
console.log(value >= 0 ? value : -value)
```
