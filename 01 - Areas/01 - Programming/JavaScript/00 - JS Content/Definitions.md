#JavaScript 

**Выражение** — это комбинация значений, переменных, операторов и функций, которая вычисляется и возвращает некоторое значение. Выражение является одной из основных единиц кода в большинстве языков программирования.

*Примеры:* 
- **Литералы:**
```JavaScript
52; 
'Hello'; 
true;
```

- **Переменные**:
```JS
let x = 10
x; // выражение, которое возвращает 10
```

- **Арифметические выражения**:
```JS
let result = 5 + 3 * 2; // вычисляется как 5 + (3 * 2) = 11
```

- **Логические выражения**:
```JS
let isAdult = age > 18; // возвращает true или false
```

### Важно!!!
Выражения можно отличать от **инструкций (statements)**. Например:

- Выражение: `5 + 3` (просто возвращает значение)
- Инструкция: `let x = 5 + 3;` (выполняет действие — создаёт переменную)
