#JavaScript
##### Примитивные:
- String
- Boolean
- Number
- null - значение отсутствует 
- undefined - значение не существует (не определено)
- symbol

##### Ссылочной тип:
-переменная хранит ссылку на область в памяти, где находится object
- object
```JavaScript
const objectA = {
a: 10,
b: true
}

const ccopyOfA = objectA
```

`copyOfA.a = 20` - изменение значения переменной `a` на 20 
`copyOfA.c = 'abc'` - добавление новой переменной `c` со значением `'abc'`
