#JavaScript 

1. 
```JS
if (условие) {
	блок кода, выполняемый однократно, если условие правдиво
}
```

2. 
```JS
if (условие) {
	блок 1
} else {
	блок 2
}
```

3. 
```JS
if (условие 1) {
	блок 1
} else if (условие 2) {
	блок 2
} else {
	блок 3
}
```
или 
```JS
if (условие 1) {
	блок 1
} 
if (условие 2) {
	блок 2
}
if (условие 3) {
	блок 3
}
```

*switch:*
```JS
switch (Выражение) {
case A:
//действие если выражение === А
break
case B:
//действие если выражение === В
break
default:
//действие по умолчанию
}
```
