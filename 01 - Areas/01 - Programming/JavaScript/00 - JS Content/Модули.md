#JavaScript

*Представляют собой файлы JS которые находятся в разных папках, при это можно экспортировать или импортировать части когда из одного файла в другой.*

Фишки:
- позволяют структурировать код 
- позволяют избегать дублирования блока кода 

##### **Пример для одного экспорта:**

файл_1:
```JS
const myName = () => {
	console.log('Nikita')
}

export default myName
```

файл_2:
```JS
import printMyName from './файл_1'

printMyName() //Nikita
```


##### **Пример для нескольких экспортов:**

файл_1:
```JS
const one = 1
const two = 'two'

export {
	one,
	two
}
```

файл_2:
```JS
import {
	one,
	two
} from './файл_1'

console.log(one) //1
console.log(two) //two
```

- [!] *имена переменных должны совпадать*

##### **Во избежание конфликта названий:**

```JS
import {
	one as oneRenamed,
	two
} from './файл_1'
```
