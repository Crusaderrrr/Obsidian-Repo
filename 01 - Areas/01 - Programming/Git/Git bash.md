This is a list of useful git bash commands.

- `cd` - change directory
- `ls` - list all files of directories of the directory (folder)
	- `ls -la` - list more info (date, time when created, etc.) 

**Files**:
- `touch` - create a file
- `nano` - open and *write* a file
- `cat` - open and *read* a file
- `cp file_name copy_file_name` - *copy* file (can move copies of files)
- `mv file_name changed_name` - *change* file name (can move file without copying)
	- `mv -t folder_name file_names*` - moving multiple files
- `rm` - *remove* file
	- `rmdir` - remove folder

**Flags**:
- `-r` - this flag stays for *recursive*, that means that the command with this flag will be executed on all the directory files.
- `-f` - stays for *force*, executing the command without confirmation.
